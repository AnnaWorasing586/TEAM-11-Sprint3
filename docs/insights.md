## Insights from User Testing

> Sprint 4 — NutriScan AI (Team 11)

เอกสารนี้สรุป Insight สำคัญที่ได้จาก User Testing ใน Sprint 4 โดยอ้างอิงจาก Feedback Summary และผลการทดสอบผู้ใช้ เพื่อนำไปใช้ในการวางแผน Prototype v2 และ Sprint 5

แต่ละ Insight ประกอบด้วย

* **Claim:** สิ่งที่ทีมค้นพบ
* **Evidence:** หลักฐานจาก Feedback และการทดสอบ
* **Implication:** สิ่งที่ทีมควรดำเนินการต่อ

---

## Insight 1 — ผู้ใช้เข้าใจ Core Flow ได้ง่าย

### Claim

ผู้ใช้สามารถทำความเข้าใจขั้นตอนหลักของ NutriScan AI ได้โดยไม่ต้องได้รับคำอธิบายเพิ่มเติมมากนัก แสดงให้เห็นว่า Core Flow ของระบบมีความชัดเจนและใช้งานได้จริง

### Evidence

* ผู้ใช้ทั้ง 5 คนสามารถดำเนินการตามขั้นตอนหลักของระบบได้
* ผู้ใช้สามารถอัปโหลดรูปฉลากอาหารได้สำเร็จ
* ผู้ใช้สามารถดูข้อมูลที่ OCR อ่านได้และเข้าถึงผลการวิเคราะห์ได้
* Feedback หลายรายการระบุว่าระบบใช้งานง่ายและเข้าใจได้ไม่ยาก

### Implication ต่อ Sprint 5

* คงรูปแบบ Core Flow เดิมไว้
* ไม่จำเป็นต้องปรับเปลี่ยนลำดับขั้นตอนหลักของระบบ
* มุ่งเน้นการปรับปรุงคุณภาพผลลัพธ์มากกว่าการออกแบบ Flow ใหม่

---

## Insight 2 — OCR Accuracy เป็นความเสี่ยงหลักของระบบ

### Claim

แม้ระบบจะสามารถอ่านข้อมูลจากฉลากอาหารได้ แต่ความแม่นยำของ OCR ยังขึ้นอยู่กับคุณภาพของรูปภาพอย่างมาก และเป็นปัจจัยที่ส่งผลต่อประสบการณ์ผู้ใช้โดยตรง

### Evidence

* ผู้ใช้บางรายพบว่าข้อมูลที่ OCR อ่านได้ไม่ครบถ้วน
* ปัญหาส่วนใหญ่เกิดจากรูปภาพไม่ชัด แสงไม่เพียงพอ หรือข้อความบนฉลากมีขนาดเล็ก
* เมื่อ OCR อ่านข้อมูลผิด ผลการวิเคราะห์ที่ตามมาก็อาจคลาดเคลื่อนได้เช่นกัน

### Implication ต่อ Sprint 5

* ยกระดับการปรับปรุง OCR เป็น Priority สูง
* เพิ่มคำแนะนำการถ่ายภาพก่อนอัปโหลด
* ปรับปรุงการตรวจสอบข้อมูลที่ OCR อ่านได้ก่อนเข้าสู่ขั้นตอนวิเคราะห์
* ทดสอบระบบกับฉลากอาหารที่หลากหลายมากขึ้น

---

## Insight 3 — ผู้ใช้ต้องการคำอธิบายผลการวิเคราะห์ที่ชัดเจนขึ้น

### Claim

ผู้ใช้สามารถดูผลลัพธ์ที่ระบบแสดงได้ แต่บางส่วนยังไม่เข้าใจความหมายของข้อมูลหรือวิธีนำไปใช้ในการตัดสินใจเลือกอาหาร

### Evidence

* ผู้ใช้บางรายระบุว่าต้องการคำอธิบายเพิ่มเติมเกี่ยวกับผลการวิเคราะห์
* ผู้ใช้เข้าใจตัวเลขข้อมูลโภชนาการ แต่ยังไม่แน่ใจว่าค่าที่แสดงควรตีความอย่างไร
* มีข้อเสนอแนะให้เพิ่มคำอธิบายหรือคำแนะนำที่เข้าใจง่ายมากขึ้น

### Implication ต่อ Sprint 5

* เพิ่มข้อความอธิบายผลลัพธ์ในหน้าแสดงผล
* ปรับปรุง UI ให้สื่อความหมายได้ชัดเจนขึ้น
* เพิ่มคำแนะนำเบื้องต้นเกี่ยวกับข้อมูลโภชนาการที่ควรระวัง

---

## Summary

| Insight                             | ระดับความสำคัญ |
| ----------------------------------- | -------------- |
| ผู้ใช้เข้าใจ Core Flow ได้ง่าย      | Medium         |
| OCR Accuracy เป็นความเสี่ยงหลัก     | High           |
| ผู้ใช้ต้องการคำอธิบายผลลัพธ์มากขึ้น | High           |
จากการทดสอบใน Sprint 4 ทีมพบว่า Core Flow ของ NutriScan AI สามารถใช้งานได้จริงและได้รับการยอมรับจากผู้ใช้ อย่างไรก็ตาม ความแม่นยำของ OCR และการสื่อความหมายของผลการวิเคราะห์ยังเป็นประเด็นสำคัญที่ต้องปรับปรุงใน Prototype v2 และ Sprint 5
