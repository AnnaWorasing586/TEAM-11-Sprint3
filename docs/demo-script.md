## 🥗 NutriScan AI Demo Script 🥩

### Introduction

สวัสดีครับ/ค่ะ พวกเราทีม 11 ขอนำเสนอโปรเจกต์ NutriScan AI

NutriScan AI เป็นระบบที่ช่วยวิเคราะห์ข้อมูลโภชนาการจากฉลากอาหาร โดยใช้เทคโนโลยี OCR และ AI เพื่อช่วยให้ผู้ใช้เข้าใจข้อมูลทางโภชนาการได้ง่ายขึ้นผ่านระบบไฟสัญญาณ 3 สี

---

### Core Flow

Core Flow ของระบบมีดังนี้

ผู้ใช้อัปโหลดรูปภาพฉลากโภชนาการอาหาร

→ ระบบ OCR อ่านข้อมูลสารอาหารจากรูปภาพ

→ ระบบวิเคราะห์ข้อมูลโภชนาการ

→ แสดงผลลัพธ์ผ่านระบบไฟสัญญาณ 3 สี พร้อมข้อมูลที่เข้าใจง่าย

---

### Demo

ตอนนี้จะสาธิตการทำงานของ Prototype v1

เริ่มจากการอัปโหลดรูปภาพฉลากโภชนาการเข้าสู่ระบบ

เมื่ออัปโหลดเสร็จ ระบบ OCR จะทำการดึงข้อมูลสารอาหารจากรูปภาพ

จากนั้นข้อมูลจะถูกส่งเข้าสู่กระบวนการวิเคราะห์

และแสดงผลลัพธ์บนหน้าจอในรูปแบบที่ผู้ใช้สามารถเข้าใจได้ง่าย พร้อมสถานะไฟสีเขียว สีเหลือง หรือสีแดง ตามระดับความเหมาะสมของสารอาหาร

---

### Current Limitations

ปัจจุบัน Prototype v1 ยังมีข้อจำกัดบางส่วน

- รองรับการใช้งานผ่าน Web Browser เท่านั้น
- OCR อาจอ่านข้อมูลผิดพลาดในกรณีที่ภาพไม่ชัดหรือมีแสงสะท้อน
- AI Recommendation บางส่วนยังใช้ Mock Data
- ยังไม่ได้ผ่านการทดสอบกับผู้ใช้จริง

---

### Sprint 4 Plan

ใน Sprint 4 ทีมจะนำ Prototype v1 ไปทดสอบกับนักศึกษา 5–10 คน

เพื่อประเมิน

- ความถูกต้องของ OCR
- ความเข้าใจในการใช้งานระบบ
- ความสำเร็จในการทำงานตาม Task
- ความพึงพอใจของผู้ใช้งาน
