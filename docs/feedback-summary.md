# Feedback Questions

## ก่อนการทดสอบ

1. คุณเคยอ่านฉลากโภชนาการก่อนซื้ออาหารหรือไม่
2. ปกติคุณใช้ข้อมูลอะไรในการเลือกซื้ออาหาร
3. คุณคาดหวังให้ระบบนี้ช่วยอะไรในการเลือกอาหาร

## หลังการทดสอบ

1. ส่วนไหนของระบบที่เข้าใจง่ายที่สุด
2. ส่วนไหนของระบบที่สับสนที่สุด
3. มีขั้นตอนไหนที่ทำได้ยากหรือไม่ เพราะอะไร
4. คุณเข้าใจผลลัพธ์ที่ระบบแสดงหรือไม่
5. หากปรับปรุงได้ 1 อย่าง คุณอยากให้ปรับอะไร
6. คุณคิดว่าจะใช้งานระบบนี้จริงหรือไม่ เพราะอะไร

## คะแนนประเมิน

- คะแนนความเข้าใจ (1-5)
- คะแนนความง่ายในการใช้งาน (1-5)
- คะแนนความน่าใช้งาน (1-5)
