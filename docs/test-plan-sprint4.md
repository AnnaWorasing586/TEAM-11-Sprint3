# Sprint 4 Test Plan

## Objective

ทดสอบ Prototype v1 ของ NutriScan AI กับผู้ใช้จริง
เพื่อประเมินความสามารถในการใช้งาน ความเข้าใจของผู้ใช้ และความถูกต้องของระบบ OCR

---

## Target Users

นักศึกษาจำนวน 5–10 คน

---

## Test Tasks

### Task 1
อัปโหลดรูปภาพฉลากโภชนาการเข้าสู่ระบบ

### Task 2
ตรวจสอบข้อมูลสารอาหารที่ระบบ OCR อ่านได้

### Task 3
ดูผลการวิเคราะห์และระบบไฟสัญญาณ 3 สี

### Task 4
ให้ความคิดเห็นเกี่ยวกับความเข้าใจและความสะดวกในการใช้งาน

---

## Observation Points

- ผู้ใช้สามารถใช้งานระบบได้เองหรือไม่
- มีจุดใดที่ทำให้ผู้ใช้สับสน
- ระยะเวลาที่ใช้ในการทำแต่ละ Task
- ปัญหาที่พบระหว่างการใช้งาน

---

## Feedback Collection Method

- Direct Observation
- Short Interview
- Google Form Questionnaire

---

## Metrics

### OCR Accuracy
เปอร์เซ็นต์ความถูกต้องของข้อมูลที่ OCR อ่านได้

### Task Completion Rate
จำนวนผู้ใช้ที่สามารถทำ Task ได้สำเร็จ

### User Satisfaction
คะแนนความพึงพอใจหลังการใช้งาน

### Usability Issues
จำนวนปัญหาด้านการใช้งานที่พบ

---

## Expected Outcomes

- เข้าใจจุดที่ผู้ใช้มีปัญหา
- ปรับปรุง UX/UI ของระบบ
- เพิ่มความแม่นยำของ OCR
- เตรียม Prototype สำหรับการพัฒนาใน Sprint ถัดไป
