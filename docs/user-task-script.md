## User Testing Script

### Introduction

สวัสดีครับ/ค่ะ

พวกเรากำลังทดสอบต้นแบบของ **NutriScan AI** ซึ่งเป็นระบบที่ช่วยอ่านข้อมูลโภชนาการจากฉลากอาหารด้วย OCR และแสดงผลการวิเคราะห์ข้อมูลโภชนาการให้ผู้ใช้งานเข้าใจได้ง่ายขึ้น

ในวันนี้ เราอยากให้คุณทดลองใช้งานระบบตามปกติ และบอกสิ่งที่คุณกำลังคิดหรือรู้สึกระหว่างการใช้งาน หากพบจุดที่สับสน ใช้งานยาก หรือมีข้อเสนอแนะเพิ่มเติม สามารถแจ้งให้ทีมทราบได้ทันที

โปรดทราบว่าระบบนี้ยังอยู่ในขั้นตอน Prototype และอาจมีบางส่วนที่ยังไม่สมบูรณ์ การทดสอบครั้งนี้ไม่ได้เป็นการประเมินผู้ใช้งาน แต่เป็นการประเมินว่าระบบของเราสามารถสื่อสารและรองรับการใช้งานได้ดีเพียงใด

ข้อมูลและความคิดเห็นที่ได้รับจะนำไปใช้เพื่อปรับปรุงระบบและใช้ประกอบการเรียนการสอนเท่านั้น

---

### User Tasks

#### Task 1 — Upload Nutrition Label

กรุณาอัปโหลดรูปภาพฉลากโภชนาการเข้าสู่ระบบ

สิ่งที่สังเกต:

* ผู้ใช้สามารถหาปุ่มอัปโหลดได้หรือไม่
* ผู้ใช้สามารถอัปโหลดรูปภาพได้สำเร็จหรือไม่

---

#### Task 2 — Review OCR Result

กรุณาตรวจสอบข้อมูลโภชนาการที่ระบบ OCR อ่านได้จากรูปภาพ

สิ่งที่สังเกต:

* ผู้ใช้เข้าใจข้อมูลที่แสดงหรือไม่
* ผู้ใช้สังเกตเห็นข้อมูลที่อ่านผิดหรือไม่

---

#### Task 3 — View Nutrition Analysis

กรุณาดูผลการวิเคราะห์โภชนาการที่ระบบแสดง และอธิบายสิ่งที่คุณเข้าใจจากผลลัพธ์

สิ่งที่สังเกต:

* ผู้ใช้เข้าใจผลลัพธ์ที่ระบบแสดงหรือไม่
* ผู้ใช้สามารถอธิบายข้อมูลที่ได้รับได้หรือไม่

---

### Post-Test Questions

1. ส่วนไหนของระบบที่คุณคิดว่าใช้งานง่ายที่สุด
2. ส่วนไหนของระบบที่คุณคิดว่าสับสนหรือใช้งานยากที่สุด
3. คุณเข้าใจผลการวิเคราะห์ที่ระบบแสดงหรือไม่
4. หากสามารถปรับปรุงระบบได้ 1 อย่าง คุณอยากให้ปรับปรุงอะไร
5. คุณคิดว่าระบบนี้มีประโยชน์ต่อการเลือกอาหารหรือไม่ เพราะเหตุใด

---

### Closing

ขอบคุณสำหรับการเข้าร่วมการทดสอบและให้ข้อเสนอแนะ

ความคิดเห็นของคุณจะช่วยให้ทีมสามารถปรับปรุง NutriScan AI ให้มีประสิทธิภาพและตอบโจทย์ผู้ใช้งานได้ดียิ่งขึ้นในอนาคต
