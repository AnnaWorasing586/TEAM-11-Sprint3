### User Testing Script

สวัสดีครับ/ค่ะ พวกเรากำลังทดสอบต้นแบบของ NutriScan AI ซึ่งเป็นระบบที่ช่วยวิเคราะห์ข้อมูลโภชนาการจากฉลากอาหาร
ในวันนี้เราอยากให้คุณลองใช้งานระบบตามปกติ และบอกความรู้สึกหรือสิ่งที่คิดออกมาระหว่างใช้งาน หากมีจุดที่สับสน ไม่เข้าใจ หรือคิดว่าสามารถปรับปรุงได้ สามารถบอกได้ทันที
ระบบในปัจจุบันยังอยู่ในขั้น Prototype และอาจมีบางส่วนที่ยังไม่สมบูรณ์ เราไม่ได้กำลังทดสอบความสามารถของผู้ใช้ แต่กำลังทดสอบว่าระบบของเราสื่อสารและใช้งานได้ดีเพียงใด
ขออนุญาตบันทึกภาพหน้าจอ รูปภาพ หรือความคิดเห็นของผู้ใช้งาน เพื่อนำไปใช้ในการปรับปรุงระบบและประกอบการเรียนการสอนเท่านั้น

---

### User Tasks

#### Task 1
ลองอัปโหลดรูปฉลากอาหารเข้าสู่ระบบ

#### Task 2
ตรวจสอบข้อมูลโภชนาการที่ระบบแสดง

#### Task 3
ดูผลการวิเคราะห์และอธิบายสิ่งที่เข้าใจ
