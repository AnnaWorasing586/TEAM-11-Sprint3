# Project Description

NutriScan AI เป็นเว็บไซต์ต้นแบบที่ช่วยวิเคราะห์ข้อมูลโภชนาการจากภาพฉลากอาหาร โดยผู้ใช้สามารถอัปโหลดรูปฉลากอาหารเข้าสู่ระบบ จากนั้นระบบจะประมวลผลข้อมูลจากภาพ วิเคราะห์ข้อมูลทางโภชนาการ และแสดงผลลัพธ์ในรูปแบบที่เข้าใจง่าย เพื่อช่วยให้ผู้ใช้สามารถตรวจสอบข้อมูลทางโภชนาการและใช้ประกอบการตัดสินใจเลือกบริโภคอาหารได้สะดวกมากยิ่งขึ้น

## ✨ ฟีเจอร์ทั้งหมด

### 🎯 Core (สแกน + วิเคราะห์)
- 📸 **Scan รูปอาหาร** ด้วยกล้องจริง (WebRTC) — Gemini Vision วิเคราะห์
- 🏷️ **Scan ฉลากโภชนาการ** อ่านค่าจากตารางจริง
- 🟦 **Scan barcode** (ZXing-js) → OpenFoodFacts database
- 📤 **Upload รูปจากแกลเลอรี** เป็น alternative
- ✏️ **แก้ไขค่า AI** ได้ถ้าวิเคราะห์ผิด

### 📊 Dashboard
- 🎯 **วงแหวนแคลอรี** + macros (โปรตีน/คาร์บ/ไขมัน)
- 💧 **บันทึกน้ำดื่ม** +100/+250/+500 ml
- 📈 **กราฟ 7 วัน** + average
- 🍽️ **รายการมื้ออาหาร** (ลบได้)
- 🤖 **AI แนะนำมื้อต่อไป** จากมาโครที่เหลือ

### 🚦 Health Rating (Traffic-light)
- 🟢🟡🔴 **สีแดง/เหลือง/เขียว** อิงเกณฑ์ UK FSA / WHO
- 🤖 **AI วิเคราะห์เฉพาะตัว** อิงจาก BMI + เป้าหมาย + มื้อที่กินไปแล้ว
- ✨ แนะนำทางเลือกที่ดีกว่า

### 👤 Account & Cloud
- 🔐 **Sign up / Login** ผ่าน Supabase Auth (email + password)
- ☁️ **Cloud sync** มื้ออาหาร/น้ำ/badges/settings ข้ามอุปกรณ์
- 🚪 ใช้งานแบบ local-only ก็ได้ (ไม่บังคับ login)

### 📈 Report
- 📊 **กราฟ 7 วันใหญ่** + adherence %
- 🏆 **อาหารที่กินบ่อย top 3**
- 🤖 **AI สรุปสัปดาห์** + tips/warnings

### ⚙️ Settings
- 👤 ชื่อ + เป้าหมายแคลอรี
- 🏥 น้ำหนัก/ส่วนสูง → BMI auto-calc + advisory
- 🎨 สีธีม 3 แบบ + **Dark mode**
- 📤 Export CSV
- 🔄 Reset ทั้งหมด

### 🎮 Gamification
- 🔥 **Streak counter** (วันติด)
- 🏆 **Badges 7 แบบ**: first scan, 10 scans, streaks 3/7/30, water goal, 3 meals/day

### 📱 PWA
- ติดตั้งบน iPhone/Android ได้
- Offline mode (cache)
- Splash screen + custom icon

### 🌐 Responsive
- มือถือ (≤500px): full-bleed
- iPad portrait: full-bleed
- iPad landscape / Notebook / Desktop: full-bleed
- รองรับ iOS safe-area (notch + home indicator)

## 🛠 Tech Stack

| Layer | เทคโนโลยี |
|---|---|
| Frontend | Vanilla HTML/CSS/JS (no framework — single page) |
| Backend | FastAPI (Python 3.12) + uvicorn |
| AI Vision | **Google Gemini 2.5 Flash** (สลับเป็น Claude Sonnet ได้) |
| Database | Supabase PostgreSQL + Row-Level Security |
| Auth | Supabase Auth (email/password) |
| Barcode | OpenFoodFacts public API + ZXing-js (browser) |
| Hosting (web) | Render Static Site (frontend) + Render Web Service (backend) |
| PWA | Service Worker + Web Manifest |

## 🚀 วิธีรัน local

### Frontend
```powershell
cd frontend/prototype
python -m http.server 5500
# เปิด http://localhost:5500
```

### Backend
```powershell
cd backend/prototype
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
$env:GEMINI_API_KEY = "AIza..."     # หรือ $env:ANTHROPIC_API_KEY
uvicorn server:app --reload --port 8000
```
